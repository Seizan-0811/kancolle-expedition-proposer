## 対応内容メモ

### 01 — 初期タブを「kc-web」に変更
- `inputTab` の初期値を `'manual'` → `'kcweb'` に変更
- タブ順を kc-web → JSONインポート → 手動入力 に変更

### 02 — ブックマークレット実行後のコピー完了通知
- コピー成功時: 隻数を含む完了 `alert` を表示
- コピー失敗時: エラー内容を `alert` で通知
- ストアが見つからない場合・shipStock が空の場合も `alert` で通知

### 03 — 登録済み艦娘リストのソート・絞り込み機能追加
- ソート選択肢: 入手順 / Lv 降順（デフォルト）/ Lv 昇順
- 艦名での部分一致絞り込み入力欄を追加（× ボタンでクリア）

### 04 — 手動入力で艦娘名未入力時の追加を防止
- 艦娘名が空白の場合「追加」ボタンを `disabled` に変更
- Firebase master.json から艦娘名を取得し、艦種に連動したサジェスト（datalist）を実装
- 手動入力の艦種選択肢に全艦種を追加（戦艦・空母・工作艦・補給艦など）

### 05 — 艦娘登録左ペインの横スクロール解消
- `.panel-left` に `overflow-x: hidden` を追加
- `.form-card` に `min-width: 0` を追加
- `.bookmarklet-wrap` に `flex-wrap: wrap` を追加
- 左右パネル間にドラッグリサイザーを実装（180px〜450px、ドラッグ中テキスト選択抑制）
