# タスク05: 艦娘登録左ペインの横スクロール解消

## 概要
「艦娘登録」の左ペインが画面の縮尺に関わらず表示幅に収まっておらず、横スクロールバーが表示されている。

## 対象ファイル
- `src/ExpeditionProposer.vue`（左ペインのスタイル定義）

## 対応内容
以下のどちらか（または両方）で対応:
1. 左ペインの `min-width` を削除または縮小し、コンテンツが幅に収まるよう調整する
2. ペイン幅をドラッグで変更できるリサイザーを実装する

まず CSS の修正（`overflow-x: hidden` や `width` 調整）で対応し、それでも解消しない場合にリサイザーを検討する。

## 優先度
中
