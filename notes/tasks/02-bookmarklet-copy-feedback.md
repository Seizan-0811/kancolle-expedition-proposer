# タスク02: ブックマークレット実行後のコピー完了通知

## 概要
ブックマークレットでクリップボードにコピーした後、何もフィードバックがないため「本当にコピーされたか？」とユーザーが不安になる。連打することで制空シミュ側への意図しないリクエスト多発につながる恐れもある。

## 対象ファイル
- ブックマークレットのコード（kc-web 上で実行されるスニペット）
- `src/ExpeditionProposer.vue`（手順説明・ブックマークレット生成部分）

## 対応内容
- ブックマークレット実行時にコピー完了のポップアップ（`alert()` または kc-web の UI に重ねた簡易通知）を表示する
- 例: `alert('艦娘データをクリップボードにコピーしました！')` をスニペット末尾に追加

## 優先度
高（UX 上の混乱防止・サーバ負荷リスク回避）
